API 文档

充易达开放 API 接口文档

📖 完整文档

接口说明

所有接口均使用 POST 请求，Content-Type 为 application/json。

请求有效期为 5分钟，需携带签名参数。

Base URL https://api.chongyida.com/v1
POST/order/create下单

创建充值订单。必传参数：userNo（用户编号）、productId（产品ID）、account（充值账号）、timestamp、sign。

POST/order/query查单

查询订单状态。必传参数：userNo、orderNo（订单号）、timestamp、sign。

POST/product/list商品列表

获取可购买商品列表。必传参数：userNo、timestamp、sign。

POST/account/info账户信息

获取账户余额等信息。必传参数：userNo、timestamp、sign。

POST/sign/generate签名生成

测试签名生成。签名算法：MD5(userNo + timestamp + AppSecret)。
